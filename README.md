**scLASER** (Single-cell *Latent Association through Spatiotemporal Expression Reconstruction*)  
is an R package for modeling, benchmarking, and visualizing spatiotemporal associations
in single-cell and spatial transcriptomics datasets.

## Installation

To install the latest development version directly from GitHub:

```r
# Install devtools if needed
install.packages("devtools")

# Install scLASER (private repo access required)
devtools::install_github("fanzhanglab/scLASER")
